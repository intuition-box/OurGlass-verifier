Independent verifier

Decode any OurGlass subscription locally and confirm what it actually authorizes — the cap, the token, the delegate — and that every enforcer is one of MetaMask's audited contracts.

This tool is deliberately separate from the OurGlass app: a different repository, hosted independently (IPFS), with the audited addresses hard-coded inside it. It calls no server and runs entirely in your browser, so a tampered OurGlass front-end cannot tamper with this check too.

Cross-check the hard-coded addresses against MetaMask's official Deployments.md. Source is public and auditable; this build is content-addressed on IPFS, so its hash is its identity.
